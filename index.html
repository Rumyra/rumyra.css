What

There's a balance here between a full blown pattern library and just a base stylesheet for demos, talks, starting things.

This is the latter, we take base HTML elements, page structure and some common components.

Everything is a seperate css file and an @import so you can include what you will. I've had to use sass to concatinate into one css file - however I have not written any sass.

I'll use atomic and bem as much as pos to try and keep things extendable for future websites & apps.
