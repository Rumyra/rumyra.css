What

There's a balance here between a full blown pattern library and just a base stylesheet for demos, talks, starting things.

This is the latter, we take base HTML elements, page structure and some common components.

Everything is a seperate css file and an @import so you can include what you will. I've had to use sass to concatinate into one css file - however I have not written any sass.

Select buttons

<div class="select--btns">
	<button data-selected="true">Original Sound</button>
	<button data-selected="false">Big Hall 1</button>
	<button data-selected="false">Big Hall 2</button>
	<button data-selected="false">Big Hall 3</button>
</div>
// quick & dirty: grab our buttons
const buttons = document.querySelectorAll('.select--btns button');

for (const butt of buttons) {
	butt.addEventListener('click', () => {
		
		// change the current button state
		const curButt = document.querySelector('button[data-selected="true"]');
		curButt.dataset.selected = 'false';
		
		// switch to our new button
		butt.dataset.selected = 'true';
		
		// functionality to do on button select
	})
}
